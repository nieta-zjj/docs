## 摘要

这是一个由 Coder Quad 和 Simplify 共同维护的 GitHub 仓库，专门为 2025 及 2026 届毕业生每日汇总美国、加拿大及远程的入门级全职科技岗位。

## 功能与定位

该项目面向应届生求职场景，核心价值是把分散在不同 ATS 系统和招聘站点的科技类岗位集中到一个持续更新的清单中，降低搜索与比对成本。仓库描述聚焦全职岗位，覆盖 SWE、Quant、PM 等方向。

## 典型使用场景

- 应届毕业生按岗位类别快速浏览在招职位。
- 求职者通过标签判断签证、国籍要求、学历要求和岗位关闭状态。
- 社区贡献者通过 GitHub Issue 提交新增岗位或修正已有岗位。

## 特色与差异点

- 每日维护更新，并给出按类别分组的岗位入口。
- 收录规则明确，优先标准化 ATS 来源（如 Workday、Greenhouse、Ashby）。
- 提供历史归档文件（`archived/README-2022.md` 到 `archived/README-2025.md`），便于回看往年数据。
- 与 Simplify 生态联动，可配合外部工具提升投递效率与更新跟踪效率。

## 使用方式概览

- 直接阅读仓库 `README.md`，按分类进入对应岗位列表。
- 新增岗位时，使用仓库 Issue 的 `New Role` 模板提交。
- 更新或关闭岗位时，使用 `Edit Role` 模板并附上目标职位链接。

## 限制与注意事项

- 当前收录范围以美国、加拿大及远程岗位为主，不是全球岗位库。
- 仓库未声明开源许可证，复用前需要自行确认使用边界。
- 岗位有效性会随时间变化，使用时应以职位原始投递页面状态为准。
