**1) 一句话总结**
Winhance 是一款基于 C# 开发的 Windows 10/11 系统精简、优化与定制工具，允许用户在不重装系统的情况下进行深度个性化设置、应用管理及部署准备。

**2) 核心要点**

* **配置同步**：支持将当前优化设置保存为配置文件，便于在其他 Windows 设备或重装后快速导入和部署。

## 功能与定位
Winhance 是一款基于 C# 开发的 Windows 增强实用程序，旨在为 Windows 10 和 11 系统提供精简（Debloat）、优化和个性化定制服务。它无需重新安装操作系统，即可实现类似 UnattendedWinstall 项目的系统增强效果。

## 典型使用场景
* **新系统初始化**：快速卸载不需要的系统自带应用，并批量安装常用的第三方软件。
* **系统深度定制**：调整系统 UI（如任务栏、开始菜单）、隐私选项及性能设置。
* **系统部署准备**：提取当前系统配置与驱动，制作自定义的 Windows 安装镜像或无人值守安装脚本。
* **多设备配置同步**：导出当前优化设置，在其他 Windows 设备上快速导入应用。

## 核心功能
* **软件与应用管理**：
  * 提供可搜索的界面，支持一键卸载或安装 Windows 内置应用、传统功能及可选功能。
  * 集成 WinGet（基于 COM API），可分类快速安装浏览器、多媒体工具等外部应用程序。
* **系统优化**：
  * 支持调整 UAC 通知级别、隐私设置、游戏与性能优化、Windows 更新策略、电源计划、声音及通知偏好。
* **界面定制**：
  * 支持切换系统深色/浅色主题，并提供针对任务栏、开始菜单和资源管理器的个性化设置选项。
* **高级部署工具**：
  * **WIMUtil**：内置 Windows 安装媒体实用程序，可创建包含当前系统驱动的自定义 Windows ISO 镜像。
  * **无人值守脚本**：可根据用户在 Winhance 中的选项，自动生成 `autounattend.xml` 文件。
* **配置管理**：
  * 支持将当前的 Winhance 及 Windows 设置保存为配置文件，以便在新系统或重装后快速导入。

## 特色与差异点
* **无需重装**：直接在现有 Windows 系统上应用各项优化和定制。
* **双模式运行**：安装程序同时提供常规安装版和便携版（Portable）。
* **独立开发与维护**：由个人开发者独立维护，非微软官方工具，且开发者倾向于独立工作（可能不会接受外部的 Pull Request）。

## 使用方式概览
* **系统要求**：支持 Windows 10 和 Windows 11（已在 Win 10 x64 22H2 及 Win 11 23H2/24H2/25H2 上完成测试）。
* **安装方式**：
  * 通过 PowerShell 执行单行命令快速下载并运行安装程序。
  * 从官网或 GitHub Releases 页面手动下载 `Winhance.Installer.exe` 运行。
