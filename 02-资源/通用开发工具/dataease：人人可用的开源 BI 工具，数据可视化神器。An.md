## 摘要

DataEase 是一款由 FIT2CLOUD 飞致云维护的开源 BI 与数据可视化工具，定位为 Tableau 的开源替代方案，强调零门槛的拖拽式分析体验与多数据源接入能力。项目支持 OLTP/OLAP、数据文件与 API 等多类数据源，提供桌面版与服务器版部署路径，并支持嵌入业务系统及 AI 问数能力（SQLBot 集成）。从公开案例看，它主要用于企业经营分析、制造与供应链可视化、以及行业软件中的嵌入式 BI 场景。

## 功能与定位

DataEase 的核心定位是“人人可用”的开源 BI 工具，目标是让业务与技术团队都能快速完成数据分析、仪表板搭建与洞察分享。

它提供拖拽式图表构建能力，降低了上手门槛，同时以开源方式提供可自部署、可二次集成的数据分析能力。

## 典型使用场景

- 企业经营分析与可视化看板建设。
- 制造业与供应链场景中的数据驱动分析与决策支持。
- 将 BI 能力嵌入低代码平台或行业软件，作为产品内置分析模块。
- 校园、医疗、物业等多行业的多场景数据展示与运营分析。

## 特色与差异点

- 以易用性为主：强调拖拽交互与低门槛使用。
- 数据源覆盖面广：支持多类数据库、文件与 API 数据。
- 部署形态灵活：同时提供桌面版与服务器版，并支持嵌入式应用。
- 开源与持续迭代：采用 GPL-3.0 协议并持续发布版本。

## 使用方式概览

- 桌面版：可直接下载安装客户端使用。
- 服务器版：支持 Linux 服务器部署，也支持 1Panel 应用商店安装方式。
- 生产环境建议：官方文档建议优先使用离线安装包方式。

## 限制与注意事项

- 服务器版快速部署需要满足基础硬件条件（README 提到建议 2 核 4G 及以上 Linux 服务器）。
- 生产环境不宜直接沿用快速体验式部署路径，应按官方生产部署建议执行。
- 项目采用 GPL-3.0 协议，二次开发与分发需遵循对应开源协议义务。
