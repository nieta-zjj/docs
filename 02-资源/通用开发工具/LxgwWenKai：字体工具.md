## 摘要

霞鹜文楷（LXGW WenKai）是一款基于 Klee One 衍生、采用 SIL OFL 1.1 协议的免费开源中文字体，提供多字重与丰富的衍生版本，旨在弥补开源中文字库中楷体和仿宋门类的稀缺。

## 功能与定位

- 提供可商用的开源中文字体，重点定位在楷体与仿宋风格的数字排版。
- 基于 Klee One 衍生并持续维护，目标是减少 DIY 补字方案在风格统一与版权上的风险。
- 通过主线与衍生分支覆盖不同设备与排版需求。

## 典型使用场景

- 应用与网站的中文字体内嵌，尤其是中短文本、注释和阅读界面。
- 需要跨简繁与多语言字符覆盖的内容产品。
- 桌面系统字体替换与开发环境字体配置，包括等宽相关分支。

## 特色与差异点

- 字汇覆盖较大：主线说明包含 CJK 基本区、扩展 A 区与韩文音节支持。
- 分支体系清晰：Lite、Screen、GB、TC、KR、Bright 与 Code 面向不同体积、字形规范和终端场景。
- 协议边界清楚：在 OFL 1.1 下可商用、可分发、可改造，但对单独售卖字体文件和保留名称使用有明确限制。

## 使用方式概览

- 直接下载 Releases 中的 TTF 文件安装使用。
- 在支持的包管理器中安装，README 提供了 Homebrew 与 Scoop 路径。
- 需要定制时可在类 Unix 环境按仓库构建流程生成字体。

## 限制与注意事项

- 该项目是开源字体，不提供传统商业授权证明；若组织流程强依赖该类证明，需要评估替代方案。
- 衍生字体需继续遵循 OFL 1.1，并遵守保留名称限制。
- 项目维护者已对部分反馈通道和模块分发策略做收敛，落地前应先核对当前 README 与 Issues 的最新说明。
