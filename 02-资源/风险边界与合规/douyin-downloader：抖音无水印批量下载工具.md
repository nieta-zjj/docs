## 摘要

### 1) 一句话总结
douyin-downloader 是一个基于 Python 的免费开源工具，支持跨平台无水印批量下载抖音视频、图集、音频及相关数据，并提供稳定版与异步增强版以满足不同场景需求。

### 2) 核心要点
* **技术栈与协议**：基于 Python 3.9+ 开发，采用 MIT 开源协议，支持 Windows、macOS 和 Linux 跨平台运行。
* **多格式无水印下载**：支持提取无水印视频（MP4）、高清图片（JPG）、原声音乐（MP3）以及相关数据（JSON）。
* **广泛的链接支持**：支持解析单视频/图集分享链接、用户主页、喜欢列表（需权限）、用户合集、音乐合集以及直播间内容。
* **双版本架构**：
  * **V1.0（稳定版）**：基于配置文件驱动，稳定可靠，完全支持单个视频下载。
  * **V2.0（增强版）**：采用异步架构，支持并发下载、智能重试、错误恢复及增量更新。
* **Cookie 自动化管理**：内置 `cookie_extractor.py`，支持通过 Playwright 自动打开浏览器扫码登录，实现 Cookie 的自动提取与保存。
* **高级下载机制**：支持多线程并发、数据库历史记录追踪，以及定期追踪博主新增作品的增量下载功能（避免重复下载）。
* **灵活的使用模式**：支持通过命令行直接传参运行，或通过编辑 `config.yml` 文件自定义下载选项（如路径、线程数、是否下载封面/音乐等）。

### 3) 风险与不足
* **V2.0 单视频下载缺陷**：V2.0 版本的单视频下载 API 目前会返回空响应，官方建议使用 V1.0 版本或改用用户主页下载来处理单视频。
* **Cookie 强依赖**：工具的正常运行强依赖于有效的账号 Cookie，一旦 Cookie 过期将导致下载失败，需要重新获取。
* **合规与版权风险**：项目声明仅供学习交流使用，禁止用于商业用途或侵犯他人权益，下载内容存在潜在的版权合规要求。

## 功能与定位
douyin-downloader 是一个免费、强大的抖音内容批量下载工具。其核心定位是帮助用户无水印地获取抖音平台上的各类媒体资源，支持跨平台（Windows、macOS、Linux）运行，并提供稳定版与异步增强版以满足不同下载需求。

## 典型使用场景
* **创作者内容备份**：批量下载指定用户主页发布的所有视频或图集。
* **素材收集**：提取无水印视频、高清图集或原声音乐（MP3）用于后期编辑参考。
* **合集与喜欢列表抓取**：批量获取特定合集内的作品，或下载账号标记为“喜欢”的内容。
* **增量更新**：定期追踪并下载特定博主的新增作品，避免重复下载。

## 核心功能
* **多格式与无水印下载**：支持下载无水印视频（MP4）、图片（JPG）、音频（MP3）以及相关数据（JSON）。
* **广泛的链接支持**：
  * **视频/图集**：单视频分享链接、直链、图集作品。
  * **用户维度**：用户主页（发布的作品）、用户喜欢的作品（需权限）。
  * **合集维度**：用户合集、音乐合集。
  * **直播**：支持直播间内容抓取。
* **Cookie 自动化管理**：内置 `cookie_extractor.py` 工具，可通过 Playwright 自动打开浏览器、支持扫码登录并自动提取/保存 Cookie；同时提供手动提取工具。
* **高级下载机制**：支持多线程并发下载、数据库历史记录追踪，以及增量下载功能。

## 特色与差异点
* **双版本并行架构**：
  * **V1.0（稳定版）**：基于配置文件驱动，经过大量测试，稳定可靠，**完全支持单个视频下载**。
  * **V2.0（增强版）**：采用异步架构，性能更优，支持并发下载、智能重试与错误恢复、增量更新，并整合了自动 Cookie 管理。
* **完全免费**：项目强调所有功能均免费提供。

## 使用方式概览
1. **环境准备**：需安装 Python 3.9+ 及相关依赖（`requirements.txt`）。
2. **配置 Cookie**：首次使用需通过内置脚本自动或手动获取抖音网页版 Cookie。
3. **执行下载**：
   * **命令行模式**：直接传入目标链接、保存路径等参数运行。
   * **配置模式**：通过编辑 `config.yml` 文件定义下载链接、保存路径、下载选项（如是否下载封面/音乐/JSON）、线程数等，随后运行主程序。

## 限制与注意事项
* **V2.0 单视频下载缺陷**：在 V2.0 版本中，单个视频下载的 API 目前会返回空响应（已知问题），官方建议改用用户主页下载或使用 V1.0 版本处理单视频。
* **Cookie 依赖**：工具强依赖有效的账号 Cookie，若 Cookie 过期会导致下载失败，需重新获取。
* **合规与版权声明**：项目仅供学习交流使用，禁止用于商业用途或侵犯他人权益，下载内容需尊重原作者版权。
