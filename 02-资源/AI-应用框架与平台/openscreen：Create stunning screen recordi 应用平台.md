## 摘要

openscreen 是一个面向产品演示与讲解视频的开源录屏应用平台，定位为 Screen Studio 的免费简化替代方案。项目强调“可用、开源、可修改、可分发”，覆盖录屏、缩放、标注、裁剪、导出等常见演示流程，适合希望以较低成本快速产出演示视频的个人与小团队。

## 功能与定位

openscreen 主要用于制作产品 Demo 与 walkthrough 视频，聚焦“基础功能完整 + 使用门槛较低”的录屏编辑体验。根据 README，其目标不是完整对标商业产品，而是在免费与开源前提下提供核心能力。

## 典型使用场景

- 为 SaaS 或应用功能制作讲解视频
- 录制产品发布说明、更新说明、功能演示
- 为教程内容制作带缩放与标注的屏幕操作视频
- 在个人项目、团队内部培训中快速产出可分享的演示素材

## 特色与差异点

- 免费且开源，可用于个人与商业场景
- 明确主打基础能力，不追求 1:1 复制商业产品的全部高级特性
- 内置录屏后处理能力，包括缩放、裁剪、标注、片段修剪与多比例导出
- 技术栈公开透明，基于 Electron、React、TypeScript、Vite、PixiJS 等构建

## 使用方式概览

- 从 GitHub Releases 下载对应平台安装包
- 安装后按系统要求授予屏幕录制与相关权限
- 在应用中完成录制、缩放与标注编辑后导出所需分辨率与比例
- 若需要二次定制，可直接基于开源代码进行修改与分发

## 限制与注意事项

- 项目当前处于 beta 阶段，可能存在稳定性与兼容性问题
- 官方说明其功能覆盖“基础需求”，不等同于商业产品的完整高级功能集
- 不同系统可能涉及额外安全权限或运行限制，需按官方说明处理
- 本仓库仅做项目定位与能力整理，不复述可能影响系统安全边界的可操作细节
