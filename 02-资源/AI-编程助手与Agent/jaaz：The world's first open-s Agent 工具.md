## 摘要

Jaaz 是一款主打本地化与隐私保护的开源多模态画布创意代理桌面应用，通过无限画布与智能 AI 代理结合，支持跨平台的图像与视频生成及编辑。项目同时支持本地模型与云端 API 混合接入，并采用社区许可证与商业许可证并行的双重许可模式。

## 功能与定位

Jaaz 定位为面向视觉内容生产的 AI 设计 Agent，强调“画布式创作 + Agent 交互 + 本地可用”。

核心定位来自仓库 README 的公开描述：开源、多模态、可本地运行，并以隐私友好为重点。

## 典型使用场景

- 用单条提示词快速生成图片或视频素材。
- 在无限画布中做故事板搭建、镜头组织与场景规划。
- 通过聊天方式执行对象插入、风格迁移与多角色一致性控制。
- 在本地模型与云端模型之间按成本、速度或隐私需求切换。

## 特色与差异点

- 本地优先：支持接入本地模型能力，降低对纯云端工作流的依赖。
- 画布式创作：将生成与编辑放在同一创作平面里，适合连续迭代。
- Agent 交互：以聊天驱动操作，不只做一次性生成。
- 跨平台桌面端：仓库构建信息显示支持 Windows、macOS、Linux。

## 使用方式概览

- 使用官网或 GitHub Releases 获取桌面端安装包。
- 启动后可直接连接云端 API 模型。
- 也可按需配置本地模型服务与自定义 API，实现混合部署。

## 限制与注意事项

- 项目采用双重许可（社区许可证 + 商业许可证）。
- 个人用户可免费使用标准功能。
- 组织在未获得商业许可证前，存在团队部署、代码修改、再分发、竞品用途等限制。
- 团队部署、二次开发、私有化与商业化场景应先确认官方商业授权条款。
