## 正文

2月22日，阿里云Coding Plan订阅服务迎来重磅更新，新增支持 Qwen3.5-Plus、Qwen3-Coder-Next、GLM-4.7、Kimi-K2.5 共4款顶尖编程模型。用户无需更改订阅和配置，即可在这些模型间自由切换。

### AI Agent 时代的算力挑战

当前，AI Agent 已成为大模型的主要应用载体，业界涌现出 Claude Code、OpenClaw 等爆火的 AI 工具。然而，Agent 在执行复杂任务、进行多轮规划与工具调用时，Tokens 消耗量呈指数级增长。数据显示，‌单个 Agent 的算力消耗是传统 Chatbot 的 100 至 1000 倍‌。

为了解决高频次编码需求场景下的成本痛点，阿里云推出了 Coding Plan 订阅服务，旨在为用户提供性价比更高、可自由选择的模型服务。

### 订阅套餐与限时优惠

阿里云 Coding Plan 目前提供 Lite 基础版和 Pro 高级版两个套餐，并已上线新用户首购 2 折的优惠活动：

*   **Lite 基础版**：每月最多可享 18,000 次请求，新用户首月仅需 7.9 元。
*   **Pro 高级版**：每月可享 90,000 次请求，足以完成数千次复杂任务，新用户首月仅需 39.9 元。

该订阅服务兼容性极强，用户可在 Qwen Code、Claude Code、Cline、OpenClaw 等主流 AI 工具中直接接入使用。

### 降低使用门槛，实现“模型自由”

此次 Coding Plan 的上新进一步降低了模型的使用成本。此前，阿里云百炼平台已上线了全新的 Qwen3.5-Plus，其 API 价格低至每百万 Tokens 0.8 元，用户仅需不到 5% 的价格即可获得媲美 Gemini 3 的高性能。

据介绍，目前阿里云百炼平台已上线 100 多款国内外主流模型，并为开发者提供了 400 多个涵盖 AI 硬件、短视频及广告内容等领域的 Agent 模板与服务，真正帮助开发者与企业享受到“模型自由”。
