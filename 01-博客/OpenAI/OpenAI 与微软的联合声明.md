## 摘要

**一句话总结**
微软与 OpenAI 发布联合声明，重申在 OpenAI 宣布新融资与新合作伙伴关系后，双方现有的核心合作条款、知识产权关系、云服务独家性及商业收入分成均保持不变。

**关键要点**
* **合作条款不变**：OpenAI 的新融资和新合作（如与亚马逊的合作）在协议预期之内，绝不改变双方此前（2025年10月）公布的合作条款。
* **知识产权（IP）关系**：微软继续保留对 OpenAI 模型和产品的独家许可和知识产权访问权。
* **商业与收入分成**：现有的收入分成安排保持不变，微软将继续分享 OpenAI 与其他云提供商合作所产生的收入。
* **Azure 的独家地位**：Azure 依然是无状态（stateless）OpenAI API 的独家云提供商；由第三方（包括亚马逊）合作产生的任何无状态 API 调用都将托管在 Azure 上。
* **第一方产品托管**：OpenAI 的第一方产品（包括 Frontier）将继续托管在微软 Azure 上。
* **AGI 定义与流程**：合同中关于通用人工智能（AGI）的定义以及确定其是否已实现的流程保持不变。
* **计算资源灵活性**：随着规模扩大，OpenAI 拥有在其他地方投入额外计算资源的灵活性（如 Stargate 星际之门项目），双方均有独立追求新机遇的空间。

## 正文

自 2019 年以来，微软（Microsoft）和 OpenAI 一直携手合作，负责任地推进人工智能的发展，并使其广泛惠及大众。最初的研究合作伙伴关系现已发展成为科技界最具影响力的合作之一——建立在相互信任、深度技术整合以及对创新的长期承诺之上。

随着围绕 AI 投资和合作伙伴关系的讨论不断增加，以及 OpenAI 宣布新的融资和新合作伙伴（[正如他们今天所做的那样](https://openai.com/index/scaling-ai-for-everyone/)），我们希望确保大家在我们现有合作框架内理解这些公告。今天的公告绝不会以任何方式改变我们之前在 [2025 年 10 月的联合博客](https://openai.com/index/next-chapter-of-microsoft-openai-partnership/)中分享的微软与 OpenAI 之间的合作条款。

**合作伙伴关系依然稳固且处于核心地位。**微软和 OpenAI 继续在研究、工程和产品开发方面密切合作，在多年深度协作和共同成功的基础上不断发展。

**我们的知识产权（IP）关系保持不变。**微软保留对 OpenAI 模型和产品的独家许可和知识产权访问权。像 OpenAI 与亚马逊（Amazon）之间的合作，一直都在我们的协议预期之内，微软很高兴看到他们共同创造的成果。

**我们的商业和收入分成关系保持不变。**现有的收入分成安排保持不变，并且一直包括分享 OpenAI 与其他云提供商合作产生的收入。

**Azure 仍然是无状态（stateless）OpenAI API 的独家云提供商。**微软是提供访问 OpenAI 模型和 IP 的无状态 API 的独家云提供商。这些 API 可以从微软购买，也可以直接从 OpenAI 购买。客户和开发者将受益于 Azure 的全球基础设施、安全性以及大规模的企业级功能。由 OpenAI 与任何第三方（包括亚马逊）合作产生的对 OpenAI 模型的任何无状态 API 调用，都将托管在 Azure 上。

**OpenAI 的第一方产品（包括 Frontier）将继续托管在 Azure 上。**

**AGI 的定义和流程保持不变。**合同中对 AGI（通用人工智能）的定义以及确定其是否已实现的流程保持不变。

**该合作伙伴关系支持 OpenAI 的增长。**随着 OpenAI 的规模扩大，它继续拥有在其他地方投入额外计算资源的灵活性，包括通过 Stargate（星际之门）项目等大规模基础设施计划。

该合作伙伴关系的初衷是为微软和 OpenAI 提供独立追求新机遇的空间，同时继续保持合作，目前两家公司都在共同和独立地践行这一点。

我们依然致力于我们的合作伙伴关系以及让我们走到一起的共同使命。我们将继续并肩工作，提供强大的 AI 工具，推进负责任的开发，并确保 AI 惠及世界各地的人们和组织。
