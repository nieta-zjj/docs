## 摘要

**1) 一句话总结**
OpenAI 与 Figma 推出基于 MCP 标准的全新集成，使产品构建者能够在 Codex 的代码环境与 Figma 的设计画布之间无缝切换，实现代码与设计双向转换的闭环工作流。

**2) 关键要点**
*   **核心集成**：Figma MCP Server 将 Codex 直接连接到 Figma 平台及其相关工具（如 Figma Make 和 FigJam）。
*   **双向工作流**：用户可以直接从 Codex 生成 Figma 设计，也可以将 Figma 文件中的设计重新实现为代码，且在整个过程中不会丢失上下文。
*   **技术标准**：该集成利用了 MCP（模型上下文协议）开源标准，使 AI 智能体能够顺畅地与外部数据源和应用程序进行交互。
*   **打破角色壁垒**：工程师可在不离开工作流的情况下进行视觉迭代，设计师无需成为全职程序员即可接近真实的代码实现。
*   **深化合作基础**：Figma 已在其平台工具中全面使用 OpenAI 模型，是 2025 年首批推出 ChatGPT 应用程序的合作伙伴之一，并在企业内部部署了 ChatGPT Enterprise。
*   **Codex 产品矩阵**：Codex 自 2025 年 1 月发布以来，已支持 CLI（命令行界面）、IDE 扩展、Web 端以及 macOS 桌面应用程序。
*   **用户增长数据**：Codex 每周活跃用户已超过 100 万，自今年年初以来使用量增长了 400% 以上。
*   **企业采用情况**：Cisco、NVIDIA、Ramp、Datadog 等大型企业以及 Harvey、Sierra 等初创公司均已为其员工采用 Codex。

## 正文

**核心要点：**

*   全新的 Codex 与 Figma 集成帮助用户在代码和设计画布之间无缝切换，从而更快地迭代和发布产品。
*   Figma MCP Server 将 Codex 直接连接到其设计平台以及 Figma Make 和 FigJam 等工具。
*   这一全新集成扩展了两家公司之间的合作伙伴关系，包括 ChatGPT 中的 Figma 应用程序，以及将最新的 OpenAI 模型引入 Figma 平台。

OpenAI 和 Figma 正在通过一项全新的代码到设计集成来深化他们的合作伙伴关系，该集成将 Figma 直接连接到 Codex。产品构建者可以轻松地从 Codex 生成 Figma 设计，并将 Figma 文件中的设计重新实现为代码。

Figma 是一个设计和产品开发平台，让团队能够实时共同创建、制作原型并迭代数字产品。MCP 是一种开源标准，使 AI 智能体（AI agents）能够与外部数据源、应用程序和工具进行交互。通过使用 MCP，Codex 与 Figma 的集成支持以更快、更顺畅的方式从代码实现过渡到设计画布中的探索。

其结果是一个全新的闭环工作流，Figma 和 OpenAI 的用户可以从任何地方开始，并在注重工艺细节的同时快速推进。无论产品理念是源于提示词、代码还是设计，Figma MCP Server 都能帮助连接最棒的创意，且在整个过程中不会丢失上下文。

Figma 首席设计官 Loredana Crisan 表示：“随着构建软件的门槛降低，创建的软件数量将呈指数级增长。现在的问题不再是你*能否*构建，而是你*构建什么*以及它如何脱颖而出。通过这种集成，团队可以将代码的优势与 Figma 无限画布带来的创造力、协作和工艺相结合，从而在他们最好的创意（而不仅仅是最初的创意）之上进行构建。”

Codex 产品负责人 Alexander Embiricos 表示：“这种集成使 Codex 对更广泛的构建者和企业来说变得更加强大，因为它不再预设你首先是‘设计师’还是‘工程师’。工程师可以在不离开其工作流的情况下进行视觉迭代，而设计师可以更接近真实的实现，而无需成为全职程序员。角色之间的界限开始变得淡化，因为系统有助于在意图与现实之间进行持续的转化。”

全新的 Codex 与 Figma 集成建立在两家公司坚实的合作基础之上。Figma 在其整个平台的工具中使用了 OpenAI 模型，并且是 2025 年首批推出 ChatGPT 应用程序的合作伙伴之一。它还在其整个组织内部推出了 ChatGPT Enterprise，以提升其员工的 AI 素养（AI fluency）。

**延续 Codex 的强劲势头**

Codex 最初于 2025 年 1 月作为 CLI（命令行界面）发布。Codex 将最强大的智能体编码模型与直观的产品体验结合在一起，随时随地满足构建者和开发者的需求——无论是通过 CLI 在终端中、作为 IDE 扩展、在 Web 上，还是最近推出的适用于 macOS 的 [Codex 桌面应用程序](https://openai.com/index/introducing-the-codex-app/)。

Codex 应用程序通过无缝的体验，实现了与智能体的多任务处理以及后台工作流的自动化。将 Figma 连接到 Codex，把代码优先工作流的构建便利性与在协作画布上进行设计的工艺结合了起来——为人们解锁了一种强大的全新方式，以构建从原型到生产级应用程序的各种内容。

全球的构建者正越来越多地将 Codex 视为协作者，贯穿从创意到执行的全过程。每周有超过 100 万人使用 Codex，自今年年初以来，Codex 的使用量增长了 400% 以上。从产品设计师到数据科学家、工程师、研究人员和产品构建者，所有人都在使用 Codex 来弥合从灵感到应用之间的差距。

Cisco、NVIDIA、Ramp 和 Datadog 等公司是为其员工采用 Codex 的全球企业，此外还包括 Harvey 和 Sierra 等初创公司。
