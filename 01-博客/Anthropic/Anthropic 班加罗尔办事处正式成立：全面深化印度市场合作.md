# Anthropic 班加罗尔办事处正式成立：全面深化印度市场合作

## 文档信息
- 来源：https://www.anthropic.com/news/bengaluru-office-partnerships-across-india

## 摘要
**1) 一句话摘要**
Anthropic 正式在印度班加罗尔启用其亚洲第二个办事处，并宣布在企业、教育、农业及公共部门等领域达成一系列合作，以全面深化其在全球第二大市场的业务布局。

**2) 关键要点**
* **市场地位与增长**：印度是 Claude.ai 全球第二大市场（近半数使用量集中在计算机和数学任务），自宣布扩张以来，Anthropic 在印度的年化营收已翻倍。
* **办事处与团队**：班加罗尔办事处是继东京之后的亚洲第二个办事处，由印度区董事总经理 Irina Ghose 领导，将专注于招聘本地人才。
* **本土语言提升**：重点优化 10 种印度常用语言的训练数据，并与本地机构合作构建针对农业和法律等场景的性能评估测试（计划公开）。
* **企业客户落地**：Cognizant 正向全球 35 万名员工部署 Claude；CRED 借助 Claude Code 将功能交付速度翻倍；初创公司 Emergent 完全基于 Claude 构建，不到 5 个月实现 2500 万美元 ARR 及 200 万用户。
* **教育领域赋能**：教育占印度总使用量的 12%。与 Pratham 合作的“随时测试机”正在 20 所学校试点（计划 2026 年底扩展至 100 所），并支持了 5000 多名“第二次机会”项目的学习者。
* **公共服务与非营利组织**：与 EkStep Foundation 合作 OpenAgriNet 农业项目；支持 Adalat AI 推出全国性 WhatsApp 法律帮助热线，提供即时案件更新和翻译。
* **MCP 标准应用**：印度统计和计划执行部（MoSPI）推出首个官方政府 MCP 服务器；Swiggy 利用 MCP 实现通过 Claude 直接订购杂货与预订餐厅。
* **其他产品动态**：发布 Claude Sonnet 4.6，并推出用于扫描代码库安全漏洞的 Claude Code Security 研究预览版。

**3) 风险/差距**
* **语言能力差距**：目前 AI 模型在英语上的表现仍优于印度的本土语言（Anthropic 正通过专项数据优化计划缩小此差距）。
* **司法服务痛点**：印度存在 5000 万件未结案的法庭案件，常规案件更新通常需要数月，且获取信息依赖纸质文件和中介（目前正通过 AI 帮助热线协助改善可及性）。

## 正文
印度目前是 Claude.ai 的全球第二大市场，这里拥有全球技术最密集的开发者社区之一。在印度，近一半的 Claude 使用量集中在计算机和数学任务上，包括构建应用程序、现代化升级系统以及交付生产级软件。

今天，随着班加罗尔办事处的正式启用，我们宣布在企业、教育和农业等多个领域达成一系列合作，进一步深化我们对印度各行业的承诺。

“印度代表着全球最具潜力的机遇之一，能够将负责任的 AI 带来的益处惠及更广泛的人群和企业，”Anthropic 印度区董事总经理 Irina Ghose 表示。“这里已经拥有卓越的技术人才、规模化的数字基础设施，以及利用技术改善人们生活的成功经验。这正是确保这项技术能够触达最需要它的人群所必需的基础。”

### 提升十亿人口的本土语言能力

印度有超过十亿人使用十几种官方认可的语言之一，但目前 AI 模型在英语上的表现仍优于其他语言。六个月前，我们在全公司范围内启动了一项计划以缩小这一差距，重点为印度使用最广泛的 10 种语言（印地语、孟加拉语、马拉地语、泰卢固语、泰米尔语、旁遮普语、古吉拉特语、卡纳达语、马拉雅拉姆语和乌尔都语）策划更高质量、更具代表性的训练数据。这不仅提升了我们的模型表现，我们也在持续努力增强这些语言的流畅度。

目前，Anthropic 正在与 Karya 和 Collective Intelligence Project 合作，并联合 Digital Green 和 Adalat AI 等印度顶尖非营利组织的领域专家，共同构建针对农业和法律等本地相关任务的性能评估测试。这项工作将指导我们如何为印度语言使用者、印度重要应用场景以及使用 Claude 的企业改进未来的模型。我们计划将这些评估工具公开，供其他人使用。

### 赋能企业、数字原生公司与初创企业

自宣布在印度的扩张计划以来，我们的年化营收已经翻倍。基于 Claude 构建解决方案的组织范围之广，充分体现了这种增长的广泛性——从大型企业、数字原生公司，到刚刚推出首款产品的初创企业。

为了支持不断增长的客户群，我们的印度团队将为这些客户提供应用型 AI 专业知识，帮助他们设计、构建和扩展量身定制的 Claude 驱动解决方案。

*   **大型企业与数字原生公司：**
    *   **Air India** 正在使用 Claude Code 帮助开发者以更快的速度和更低的成本交付定制软件，这也是其在运营中全面推广智能体 AI（Agentic AI）战略的一部分。
    *   **CRED** 借助 Claude Code 实现了功能交付速度翻倍，测试覆盖率提升了 10%。
    *   **Cognizant** 正在向全球 35 万名员工部署 Claude，以现代化升级遗留系统、加速软件开发，并支持其企业客户采用 AI。
*   **初创企业：**
    *   **Razorpay** 已将 AI 整合到全公司的风险系统、决策流程和日常运营中。
    *   **Rocket** 利用 Claude 让企业内的非技术团队能够在几分钟或几小时内（而非数周）构建出生产级应用和网站。
    *   **Enterpret** 的 AI 助手由 Claude 驱动，其工程团队每天使用 Claude Code 进行开发，并且该初创公司已发布了 MCP 集成，将客户洞察直接引入 Claude。
    *   **Emergent** 是一个由 AI 驱动的平台，允许任何人通过自然语言描述来构建软件。该平台完全基于 Claude 构建，在不到五个月的时间内就达到了 2500 万美元的年度经常性营收（ARR）和 200 万用户。

### 助力低收入社区教育

教育和教学任务占印度 Claude.ai 总使用量的 12%。印度最大的教育非营利组织之一 **Pratham** 选择 Anthropic 作为其首个战略 AI 实验室合作伙伴，正是因为我们对安全性与教育严谨性的共同关注。他们由 Claude 驱动的“随时测试机（Anytime Testing Machine）”目前正在 20 所学校的 1500 名学生中进行试点，并计划到 2026 年底扩展到 100 所学校。今年早些时候，该系统还被调整用于支持 Pratham 的“第二次机会（Second Chance）”项目中的 5000 多名学习者（主要为辍学女性），旨在通过帮助学生进行考试练习，创造灵活、可靠的学习和认证途径。

此外，Anthropic 正在与 **Central Square Foundation** 合作，探索如何更有效地利用教育科技和 AI 来教育来自欠发达社区的儿童。作为合作的一部分，Anthropic 将为开发 AI 工具（包括个性化辅导、教师培训解决方案和评估驱动教学）的组织提供技术专长、指导和 API 额度，目标是惠及印度各地更多的小学生。

### 将 AI 引入公共部门与非营利组织

印度在构建改善民生的互操作性数字公共基础设施方面拥有丰富的经验。Anthropic 正在与 **EkStep Foundation** 合作，探索 AI 如何在这些努力的基础上，在对印度最重要的领域产生人口规模的影响。农业就是一个典型例子——它占据了印度经济的近六分之一，并雇佣了近一半的劳动力。通过 OpenAgriNet 项目，我们正致力于部署 Claude，以扩大这一关键领域的专家知识获取渠道。

我们还在展示 Claude Code 和 Cowork 如何在非营利组织内部发挥作用，包括为数百万家庭提供无障碍健康指导的 **Noora Health**，以及将偏远社区患者与优质医疗服务连接起来的 **Intelehealth**。

印度目前有 5000 万件未结案的法庭案件，常规的案件更新通常需要数月才能传达给诉讼当事人。获取案件信息往往需要多次前往法院或通过中介来处理纸质文件和法律术语。Anthropic 正在支持 **Adalat AI** 改善司法服务的可及性，并于今日推出了一条全国性的 WhatsApp 帮助热线。借助 Claude，该热线能够以印度本土语言提供即时的案件更新、翻译、文档摘要以及法律文件的交互式查询。

### 通过开源标准推动 AI 普及

Anthropic 创建了模型上下文协议（MCP），作为连接 AI 应用程序与外部系统的通用开源标准，并于近期将其捐赠给了 Linux 基金会。

在非营利组织 Bharat Digital 的支持下，印度统计和计划执行部（MoSPI）最近推出了首个印度官方政府 MCP 服务器，使 AI 系统的用户能够以开放和互操作的方式访问和查询权威的国家统计数据。在私营部门，**Swiggy** 利用 MCP 让用户能够直接通过 Claude 订购杂货和预订餐厅。

### 扩大在印度的业务版图

随着我们在印度业务的扩大，这些合作关系将在未来几个月和几年内继续深化。我们全新的班加罗尔办事处现已正式启用，这也是 Anthropic 继东京之后的第二个亚洲办事处。该办事处由企业和初创技术领域的资深领导者、印度区董事总经理 Irina Ghose 领导，将专注于在广泛的岗位上招聘本地人才。
