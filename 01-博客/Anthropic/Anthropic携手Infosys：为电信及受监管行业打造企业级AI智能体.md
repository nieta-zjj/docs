# Anthropic携手Infosys：为电信及受监管行业打造企业级AI智能体

## 文档信息
- 来源：https://www.anthropic.com/news/anthropic-infosys

## 摘要
**1) 一句话总结**
Anthropic与Infosys达成战略合作，将Claude模型与Infosys Topaz平台深度整合，共同为电信、金融等受监管行业开发和交付企业级AI智能体解决方案。

**2) 关键要点**
*   **技术整合**：Anthropic的Claude模型和Claude Code将与Infosys Topaz平台结合，帮助企业在满足合规与透明度要求的前提下加速软件开发。
*   **印度市场战略**：印度是Claude.ai全球第二大市场，近半数使用量集中于高强度技术工作；Infosys是Anthropic在印度扩张的首批重要合作伙伴之一。
*   **聚焦智能体AI（Agentic AI）**：双方将利用Claude Agent SDK等工具，构建能独立处理多步骤复杂任务（如理赔、代码生成、合规审查）的AI智能体，并助力遗留系统现代化改造以降低成本。
*   **行业定制化方案**：
    *   *电信业*：实现网络运营现代化与智能自动化。
    *   *金融服务*：加速风险评估、自动化合规报告及提供个性化客户互动。
    *   *制造与工程*：加速产品设计与模拟，缩短研发周期。
    *   *软件与运营*：使用Claude Code编写和调试代码，利用Claude Cowork自动化日常文档和报告处理。
*   **云平台支持**：Claude目前是唯一在Amazon Bedrock、Google Cloud Vertex AI和Microsoft Azure三大全球领先云平台上均可使用的前沿模型。
*   **近期其他动态**：Anthropic发布了具备前沿性能的Claude Sonnet 4.6，推出了处于有限研究预览阶段的Claude Code Security（用于扫描代码漏洞），并与卢旺达政府签署了医疗与教育领域的AI合作备忘录。

**3) 风险/差距**
*   **落地应用差距**：在演示环境中运行的AI模型与在受监管行业中实际落地的模型之间存在巨大鸿沟，必须依靠深厚的领域专业知识才能弥合。
*   **代码安全隐患**：传统的代码审查方法经常会遗漏安全问题（Anthropic正通过研发Claude Code Security来针对性解决此漏洞）。

## 正文
Anthropic今日宣布与总部位于班加罗尔的全球下一代数字服务与咨询领导者Infosys达成合作。双方将共同为电信、金融服务、制造和软件开发等行业开发并交付企业级AI解决方案。

此次合作将Anthropic的Claude模型和Claude Code与Infosys Topaz（一套基于生成式和智能体AI技术的AI优先服务、解决方案和平台）深度整合，旨在帮助企业加速软件开发，并在满足受监管行业所需的治理和透明度要求下采用AI技术。

### 印度市场的战略地位

印度目前是Claude.ai的全球第二大市场，拥有极其活跃的开发者社区。在印度，近一半的Claude使用量集中在构建应用程序、系统现代化改造以及发布生产级软件等高强度的技术工作上。Infosys正是Anthropic在印度扩大业务版图的首批重要合作伙伴之一。

Anthropic首席执行官兼联合创始人Dario Amodei表示：“在演示环境中运行的AI模型与在受监管行业中实际落地的模型之间存在巨大鸿沟。要弥合这一鸿沟，必须具备深厚的领域专业知识。Infosys在电信、金融服务和制造等关键行业恰好拥有这种专长。他们的开发者已经在使用Claude Code来加速工作，并为那些要求高精度、合规性和深厚领域知识的行业创建AI智能体。”

Infosys首席执行官Salil Parekh指出：“AI不仅在改变业务，更在重新定义行业的运营和创新方式。我们与Anthropic的合作是推进企业级AI的战略性飞跃，将使组织能够释放价值，变得更加智能、更具韧性且更负责任。从利用智能风险管理和合规性实现金融服务现代化，到赋能工程企业以AI驱动设计和制造，我们的目标是利用双方的专业知识，加速全球企业的AI价值转化。”

### 为受监管行业构建智能体AI

此次合作的核心重点是“智能体AI（Agentic AI）”——这类系统不仅能回答问题，还能独立处理多步骤任务，如处理索赔、生成和测试代码或管理合规性审查。

借助Claude Agent SDK等工具，Infosys和Anthropic将帮助客户构建能够在漫长且复杂的任务和流程中持续工作的AI智能体。此外，双方还将结合Infosys Topaz和Claude，帮助组织对遗留系统进行现代化改造，使迁移过程更快、更轻松，并降低更新老化基础设施的成本。

双方将共同开发针对特定行业的定制化AI智能体：

*   **电信业**：AI智能体将帮助运营商实现网络运营现代化、简化客户生命周期管理并改善服务交付，为这个全球运营最复杂且受高度监管的行业之一引入智能自动化。
*   **金融服务**：帮助金融机构更快速地检测和评估风险，自动化合规报告，并提供更个性化的客户互动（例如根据客户的完整账户历史和市场状况量身定制财务建议）。
*   **制造与工程**：Claude将协助加速产品设计和模拟，缩短研发周期，使工程师能够在投入生产前测试更多迭代版本。
*   **软件开发**：开发团队将使用Claude Code来编写、测试和调试代码，从而加快从设计到生产的流程。
*   **企业运营**：Claude Cowork将帮助团队自动化处理日常工作，如文档摘要、状态报告和审查周期。

### 平台可用性

目前，Claude是唯一一款在三大全球领先云服务平台上均可使用的前沿模型，包括：Amazon Bedrock、Google Cloud Vertex AI 和 Microsoft Azure。

### 相关动态速览

*   **Claude Code Security**：这项内置于网页版Claude Code的新功能目前已进入有限的研究预览阶段。它能够扫描代码库中的安全漏洞，并为人工审查提供针对性的软件补丁建议，帮助团队发现和修复传统方法经常遗漏的安全问题。
*   **医疗与教育领域的AI应用**：Anthropic已与卢旺达政府签署谅解备忘录（MOU），共同推进AI在医疗和教育领域的应用。
*   **Claude Sonnet 4.6发布**：全新推出的Sonnet 4.6在编程、智能体应用和大规模专业工作方面展现了前沿的性能表现。
