# LocalStack 放弃 AWS 社区版引发开发者担忧

## 文档信息
- 来源：https://www.infoq.cn/article/Ek2SwBQSUmCAbg9V7WWD?utm_source=rss&utm_medium=article

## 摘要
**1) 一句话总结**
LocalStack 宣布放弃开源的 AWS 社区版，转而采用需要注册的单一镜像分发模式，此举引发了开发者社区对工作流中断及定价模式的担忧。

**2) 关键要点**
*   **版本合并**：LocalStack 将原有的开源社区版（Apache 2.0 许可证）与付费专业版合并为一个需要注册的单一镜像。
*   **分发与时间节点**：切换计划将于 3 月进行，新版本将通过 Docker Hub 上的单一镜像分发，并引入新的 CLI v2。
*   **官方动机**：官方表示，随着维护高保真 AWS 仿真的复杂性和安全要求增加，需要新的分发模型来直接与用户互动并维持可持续投资。
*   **代码库状态**：旧版社区版的源代码仍保留在 GitHub 上，但该存储库将处于非活动状态，且不再发布任何更新或安全补丁。
*   **定价方案**：提供面向个人娱乐使用、学生和开源项目的免费计划；付费计划起价为每个许可证每月 39 美元（按年计费）。
*   **CI 积分争议**：免费计划不包含 CI 积分，引发社区质疑，官方开发者关系负责人承认该问题并表示可能很快会进行修订。
*   **社区替代方案**：受此变更影响，社区开发者开始探讨使用 Moto（AWS 模拟库）、Vera AWS（本地 EC2 模拟器）或利用 AI 智能体构建替代工具。

**3) 风险/缺口**
*   **工作流中断风险**：目前自动从 Docker Hub 拉取 `localstack/localstack` 最新社区镜像的项目，必须在变更生效前更新其工作流程，否则将受到影响。
*   **安全与维护风险**：社区版将不再获得任何产品增强和安全补丁，继续使用旧版镜像将面临缺乏安全维护的风险。
*   **CI 构建限制**：由于免费计划不包含 CI 积分，依赖 LocalStack 进行 CI 构建的免费用户将面临功能受限的缺口。

## 正文
LocalStack 最近宣布对其 AWS 云模拟器的交付方式进行重大更改，正式放弃了广受欢迎的开源社区版，转而创建了一个需要注册的单一镜像。这一变动意味着，目前使用最新社区镜像的项目将需要更新其工作流程。

### 交付方式的重大变更

在过去的几年里，LocalStack 为 AWS 维护了两个独立的版本：
*   **社区版**：根据 Apache 2.0 许可证开源，对社区免费提供。
*   **专业版**：只能通过付费许可证访问。

根据最新公告，LocalStack 现在将这些组件合并为一个单一镜像。新模式将为个人和开源用户提供一个免费的、基于账户的选项，并引入了一个新的命令行界面（CLI v2）以支持本地云开发。

### 官方回应：为何做出改变？

LocalStack 的联合创始人兼联合首席执行官 Waldemar Hummer 和 Gerta Sheganaku 对此次变更解释道：

> “LocalStack 最初是一个简陋的开源实验，是社区使其成为今天的样子。然而，随着时间的推移，维护高保真 AWS 仿真的范围、安全要求和运营复杂性显著增加。为了继续提供准确、安全和生产级别的云仿真——同时仍然提供免费的入口点——我们需要一个分发模型，让我们能够直接与用户互动，了解 LocalStack 的使用情况，并可持续地对平台进行投资。”

### 社区反响与替代方案

LocalStack 是一款流行的云服务模拟器，它允许应用程序在不连接到实时 AWS 云的情况下，在本地机器上进行开发和测试。此次变更在 Reddit 等社区引发了广泛的讨论与担忧。

虽然一些开发者曾希望 AWS 有朝一日能收购 LocalStack，但不少用户对官方的声明表达了不满。用户 alvsanand 评论道：
> “看到他们称其为‘开源实验’而不是一个完整的项目，真是讽刺，尤其是因为他们的整个声誉是建立在开源之上的。他们有权这样做，但他们不应该通过假装这样做来侮辱我们的智商。”

面对这一变化，社区也在积极探讨应对策略。用户 rad15h 建议开发者可以利用 AI 智能体自己构建替代工具。此外，从业者们还提到了以下替代方案：
*   **Moto**：一个允许模拟 AWS 服务的库。
*   **Vera AWS**：一个近期推出的本地 EC2 模拟器。

### 对现有用户的影响与后续计划

展望未来，LocalStack 将不会再发布任何针对社区版的更新，所有的产品增强和安全补丁都将只应用于新版本。

**工作流更新警告**
Hummer 和 Sheganaku 警告称，对于目前使用 LocalStack For AWS 社区版（即 `localstack/localstack` Docker 镜像）的用户，任何自动从 Docker Hub 拉取最新镜像的项目，都需要在变更生效前进行更新。

**时间节点与代码库状态**
切换计划将在 3 月进行，新的 LocalStack for AWS 将通过 Docker Hub 上的单一镜像分发。社区版以前版本的源代码仍然可以在 GitHub 上获取，但该存储库将处于非活动状态。

**定价与 CI 积分争议**
*   **免费计划**：对于“娱乐性”使用 AWS 的开发者提供有限的免费计划，该工具对学生和开源项目仍然免费。
*   **付费计划**：在 AWS 上的付费计划起价为每个许可证每月 39 美元（按年计费）。
*   **CI 积分问题**：分配给工作区的 CI 积分数量取决于定价层级，而免费计划不包括这些积分。许多 Reddit 开发者质疑这种 CI 构建的积分系统“不合理”。对此，LocalStack 的开发者关系负责人 Brian Rinaldi 承认了这一挑战，并表示公司可能很快就会对其进行修订。
